Review packets — the evidence record for each task; the PR links its packet. Start from
`../templates/review.md`. A closed review packet is *transitory* — its durable record is the
finding, the spec, and the merged PR — so once the task closes it can age out to git history or
`archive/` on a 30–90-day window (see `../archive/README.md`, Suspec ADR-0096).
