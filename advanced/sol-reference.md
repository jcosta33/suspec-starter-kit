# SOL — structured requirements reference card

SOL is the optional stricter spec surface, selected per file with `format: sol` in the spec's
frontmatter. The notation is single-sourced in the Corpus repo — see
[`docs/reference/structured-requirements.md`](https://github.com/jcosta33/corpus/blob/main/docs/reference/structured-requirements.md).
This kit does not duplicate it.
